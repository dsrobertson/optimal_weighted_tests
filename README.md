# optimal_weighted_tests
R code for the paper "Optimal weighted tests for replication studies and the 'two-trials rule' with multiple hypotheses"

The files in the `examples` folder can be used to replicate the two illustrative examples given in Section 3.

The file `heatmap/run_sim.R` can be used to replicate Figures 1-4 in Section 4.2, as well as Figures C.8 and C.9 in the Appendix.

The file `run_sim.R` can be used to replicate Figures 5 and 6 in Section 4.2, Table 5 in Section 4.3, and Figures C.1-C.7 in the Appendix.

The file `compare/run_compare_sim.R` can be used to replicate Tables 4 in Section 4.2, as well as Tables 9-10 in the Appendix.

The file `correlated/run_sim.R` can be usd to replicate the Figures given in the Supplementary Materials.
